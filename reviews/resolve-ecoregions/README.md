# Ecoregions

[Ecoregions2017](https://www.resolve.ngo/our-work/biodiversity-wildlife/) from RESOLVE, based on Dinerstein et al. (2017) BioScience.

## Why we use it
- **Biome classification** — 14 terrestrial biomes for ecosystem context
- **Realm context** — 8 biogeographic realms
- **Spatial joins** — 846 ecoregion polygons for city-level ecosystem typing

## Data
- 846 ecoregions across 14 biomes and 8 realms
- Formats: Shapefile (SHP). Sample: GeoPackage.

## Current approved release
**2017**
