Sample CSV files for exploration and validation. The full microdata (`.sav`, ~1.3 GB) is not in this repository.

| File | Description |
|------|-------------|
| casen_2022.csv | Person-level records with selected columns (folio, region, expr, ytotcorh, pobreza, activ, o28c) |
| location_id.csv | Region/comuna codebook (CUT codes and names) |
| curated_data.csv | Aggregated indicators by comuna (attribute_type, attribute_value, attribute_units) |
| cepal_income_variables.csv | Income and poverty variable codebook |
| modulo_trabajo_variables.csv | Work module variable codebook |

To reproduce the sample from source: download CASEN 2022 microdata from [Observatorio Social](https://observatorio.ministeriodesarrollosocial.gob.cl/encuesta-casen)
