# UN/LOCODE (UNECE CEFACT)

[UN/LOCODE](https://unece.org/trade/cefact/UNLOCODE-Download) (United Nations Code for Trade and Transport Locations) is a global code list maintained by UNECE for ports, airports, rail terminals, and other trade/transport locations.

## Why we use it
- **Location codes** — Join city/location lists via locode (e.g. `CL SCL` for Santiago)
- **Geocoding** — Coordinates (lat/lon) for many locations
- **Transport functions** — Port, airport, rail, etc.
- **Subdivision links** — ISO 3166-2 subdivision codes

## Data
- Code list in 3 parts (Part1/2/3 CSV); encoding typically Latin-1
- Columns: country, locode, name, name_wo_diacritics, coordinates, function, status, etc.
- Updated 2–3 times per year
