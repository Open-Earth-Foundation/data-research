# Sample data — UN/LOCODE 2024-2

Code list parts from the full [UNLOCODE download](https://unece.org/trade/cefact/UNLOCODE-Download).

| File | Description |
|------|-------------|
| 2024-2 UNLOCODE CodeListPart1.csv | Part 1 (Countries A–E) |
| 2024-2 UNLOCODE CodeListPart2.csv | Part 2 (Countries F–L) |
| 2024-2 UNLOCODE CodeListPart3.csv | Part 3 (Countries M–Z) |
| 2024-2 SubdivisionCodes.csv | ISO 3166-2 subdivisions with type (Parish, Province, City, Region, etc.) |

**Encoding:** Latin-1 (ISO-8859-1). Specify `encoding='latin-1'` or `encoding='ISO-8859-1'` when reading.

**Columns:** change, country, locode, name, name_wo_diacritics, subdiv, status, function, date, iata, coordinates, remarks
