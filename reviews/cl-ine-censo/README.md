# Chile Population & Housing Census (INE)

Official national census dataset published by Chileâ€™s National Statistics Institute (INE).

## Why we use it
Primary source for:
- population and household structure
- housing characteristics
- employment by sector
- infrastructure access (electricity, etc.)
- vehicle ownership (proxy for mobility/emissions)

Key input to:
- City context indicators
- Socioeconomic vulnerability

## Current approved release
**2024**

## Known strengths
- Official national statistics
- Full territorial coverage
- High spatial resolution (down to census block)
- Large variable set (~189 variables)

## Known limitations
- Census periodic (not annual)
- Some indicators released in stages
- Definitions can change between censuses
- API access limited (bulk download/manual)
